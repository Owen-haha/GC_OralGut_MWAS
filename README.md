# GC_OralGut_MWAS
Key scripts for oral (saliva + tongue) and gut (stool) metagenome-wide association studies (MWAS) in gastric cancer.
MWAS was run on species and pathway levels.

The corresponding publication can be found at https://doi.org/10.1016/j.xcrm.2026.102761 (Distinct signatures in the human gut and oral microbiomes of gastric cancer).
